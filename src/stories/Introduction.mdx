# tmux 완벽 가이드

**터미널 멀티플렉서 tmux와 Claude Code Agent Team 활용법**

---

## 이 가이드에 대해

이 문서는 Linux 환경에서 tmux를 효과적으로 활용하는 방법과, tmux를 기반으로 Claude Code의 Agent Team 기능을 활용하는 방법을 다룹니다.

## 구성

### 1부: Linux tmux 사용 가이드

tmux의 기본 개념부터 실전 활용까지 단계별로 안내합니다.

- tmux란 무엇인가
- 설치 및 기본 설정
- 세션, 윈도우, 패인 관리
- 키 바인딩과 커스터마이징
- 실전 워크플로우와 팁

### 2부: tmux로 Claude Code Agent Team 활용하기

tmux 환경에서 Claude Code의 멀티 에이전트 기능을 최대한 활용하는 방법을 소개합니다.

- Claude Code Agent Team 소개
- tmux 기반 에이전트 팀 구성
- 실전 프로젝트 워크플로우
- 팁과 베스트 프랙티스
